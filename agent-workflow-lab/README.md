# Agent Workflow Lab

A small repo for trying pi-mono with a local LM Studio model.

## Try The App

```bash
npm start -- add "Try pi"
npm start -- add "Run tests"
npm start -- list
npm start -- done 1
npm start -- summary
npm test
```

## First pi Task

From this directory, run:

```bash
pi
```

Then ask:

```text
Add a `clear-done` command to remove completed tasks. Update the usage text and tests.
```
