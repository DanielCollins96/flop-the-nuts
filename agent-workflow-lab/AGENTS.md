# Agent Instructions

This is a small practice repo for learning pi-mono agent workflows.

## Workflow

1. Read the existing code before editing.
2. Keep changes small and focused.
3. Run `npm test` after code changes.
4. Prefer simple Node standard-library APIs. Do not add dependencies unless asked.

## Project

The app is a tiny task-list CLI. It stores tasks in `tasks.txt` using this format:

```text
[ ] Open task
[x] Completed task
```

## Suggested First Agent Task

Ask pi:

```text
Add a `clear-done` command to remove completed tasks. Update the usage text and tests.
```
